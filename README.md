\# HMM - Baum Welch Algorithm



Name: Shobin Pn  

Register Number: TCR24CS062  



\## Project Description

This project implements the Hidden Markov Model using the Baum-Welch Algorithm (Expectation Maximization).



\## Inputs

\- Observed state sequence

\- Number of hidden states



\## Outputs

\- Transition Matrix (A)

\- Emission Matrix (B)

\- Initial Distribution (π)

\- Probability P(O | λ)

\- State Transition Diagram

\- Hidden State Probability Graph



\## How to Run



1\. Install Python

2\. Install required libraries:



pip install numpy matplotlib streamlit graphviz



3\. Run the app:



python -m streamlit run app.py



4\. Open the browser at:

http://localhost:8501
